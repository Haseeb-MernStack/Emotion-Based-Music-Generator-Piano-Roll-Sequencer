Thank you for your interest in contributing to Music Composer!

How to contribute

1. Fork the repository and create a feature branch.
2. Run `npm install` and make your changes.
3. Add tests where appropriate.
4. Open a pull request describing your changes.

Code style

- Follow TypeScript and React conventions used in the repo.
- Run `npm run lint` (if available) before creating a PR.

Security

- If you find a security issue, please open a private issue or contact the maintainers.
